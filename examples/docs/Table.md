    <infinite-table
      :table-data="tableData"
    />
  </el-row>

### Columns 的 options

| 参数   | 说明                        | 类型        | 默认值 |
| ----   | -------------------- | ----------- | --------------------------- |
| name   | 表格列title           | _string_    |                             |
| fixed  | 该列是否为固定列       | _boolean_   | false                       |
| isSort | 是否支持排序          | _boolean_    | 固定列为false，非固定为true  |
| sort   | 列排序方式            | _string_     | 1正序 0倒序                 |
| key    | 表格列key             | _string_    |                             |
| unit   | 表格列单位             | _string_    |                             |
| style  | 表格列补充样式         | _object_    |                             |
